# Macropad
# 3x3 Macropad with Rotary Encoder

## Overview

This is a custom 3x3 macropad designed for productivity and media controls. It uses a Seeed XIAO RP2040, a rotary encoder with a push button, a custom PCB designed in KiCad, and a custom case designed in Fusion 360.
## Features

- 9 programmable keys
- Rotary encoder for volume control
- Encoder push button for mute
- Custom PCB
- Custom 3D-printed enclosure
- KMK firmware

- Press: Mute/Unmute
